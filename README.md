# Call_Center_Analysis-Dashboard
This Power BI dashboard analyzes call center performance, agent productivity, and customer satisfaction metrics.

# Business Problem
Call centers generate large volumes of customer interaction data. Without proper analysis, it becomes difficult to identify performance bottlenecks, monitor agent productivity, and improve customer satisfaction.

# Key KPIs
Total Calls
Resolved Calls
Unresolved Calls
Average Handling Time
Customer Satisfaction Score
Agent Efficiency Metrics

# Key Insights
Identified high-performing agents.
Highlighted peak call volume periods.
Analyzed customer satisfaction trends.
Revealed opportunities to improve resolution rates.
